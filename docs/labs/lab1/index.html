Тема работы: Знакомство с Android Studio, создание простейшего приложения

Цель работы:

  • ознакомиться с инструментами разработки Android-приложений;
  • научиться запускать приложение на эмуляторе;
  • ознакомиться с работой Activity и созданием макета;
  • изучить вопрос доступа к объектам UI в классе Activity;
  • научиться осуществлять простейшую обработку событий.

Задание.

Необходимо разработать простое приложение. Пользователь вводит свое имя, нажимает на кнопку и появляется приветственная надпись. При смене языка с английского на русский, приложение должно изменить язык надписей на русский.
  1. Установите Java Development Kit - https://www.oracle.com/java/technologies/javase-downloads.html
  2. Установите Android Studio по следующей ссылке - https://developer.android.com/studio

При установке Android Studio выделите компонент Android Virtual Device

  1. Убедитесь, что ваша операционная система и ваш процессор удовлетворяет требованиям для запуска эмулятора Android-устройства. При необходимости, установите дополнительное ПО. Подробности здесь.
  1. При запуске Android Studio выберите пункт меню Configure, после чего выберите SDK Manager

  1. Во вкладке SDK Platform убедитесь, что у вас загружены библиотеки для актуальной версии Android

  1. Во вкладке SDK Tools убедитесь, что у вас загружены следующие инструменты SDK: Android SDK Build-Tools, Android Emulator, Android SDK Platform Tools, HAXM Installer для процессоров Intel или Hypervisor Driver for AMD для процессоров AMD, Layout Inspector for API 29-30. Если вы планируете тестировать приложение на физическом устройстве, установите также Google USB Driver.

  1. Нажмите OK, после чего Android Studio установит требуемые инструменты.
  1. В стартовом окне Android Studio выберите пункт Create New Project.

  1. В разделе Select a Project Template выберите Empty Activity и нажмите Next.

  1. В разделе Configure Your Project укажите название проекта, путь к проекту, язык и минимальную версию ОС для запуска вашего проекта (рекомендуется установить версию API 23). Чекбокс Use legacy android support libraries выделять не следует. После указания параметров проекта, нажмите Finish.

  1. Дождитесь завершения генерации проекта.
  1. Запустите AVD Manager (с помощью кнопки на тулбаре или в меню Tools -> AVD Manager) и создайте новое виртуальное устройство.

  1. Выберите подходящие физические параметры устройства, после чего нажмите Next.

  1. В разделе System Image выберите актуальную версию операционной системы. Убедитесь, что вы устанавливаете версию x86, после чего нажмите Next.

  1. Далее вы можете изменить параметры устройства (настройки по-умолчанию вполне подходят), а также рекомендуется убрать скин для виртуального устройства). После всех настроек нажимайте Finish.

  1. Закройте окно AVD Manager и запустите ваш проект.

  1. Дождитесь загрузки эмулятора и запуска вашего приложения.

  1. Убедитесь, что эмулятор работает с приемлемой скоростью, отсутствуют существенные подтормаживания и задержки. Если ваш эмулятор работает медленно, убедитесь, что вы корректно настроили ваш компьютер для эмуляции android-устройства.

Для начала нам необходимо создать макет нашего простейшего приложения.

  1. Откройте макет res\layout\activity_main.xml

  1. Выделите и удалите виджет TextView, который был добавлен при генерации проекта, после чего найдите в палитре виджетов (Palette) Text -> Plain Text (это виджет поля ввода, класс EditText), добавьте его на экран, после чего привяжите виджет ко всем четырем сторонам экрана.

  1. Добавьте виджет TextView (Text -> TextView). Эта текстовая надпись должна находиться сверху от поля ввода с выравниванием по левой стороне. Расстояние между верхней границей поля ввода и нижней границей текстовой надписи должно составлять 8dp.

Обратите внимание, что мы задумали такой макет, что левая граница текстовой надписи должна быть привязана к левой границе поля ввода. Для этого левую границу текстовой надписи мы буквально привязываем ограничением к левой границе поля ввода. Теперь, если поле ввода сместится по горизонтали, то поле ввода также сместится, чтобы соблюдалась привязка.

Далее, чтобы текстовая надпись располагалась вверху от поля ввода, нижнюю границу текстовой надписи мы привязываем к верхней границе поля ввода. Если поле ввода сместится по вертикали, то текстовая надпись также сместится таким образом, чтобы соблюдать ограничение.

Если мы хотим сделать отступ между нижней границей надписи и верхней границей поля ввода, мы просто указываем отступ в панели атрибутов, как показано на скриншоте выше.

  1. Теперь добавим еще одну текстовую надпись. Она должна располагаться снизу от поля ввода на расстоянии 16dp и должна быть отцентрирована по отношению к полю вводу. Также сделаем эту надпись больше, для этого необходимо изменить параметр textAppearance.

  1. Добавим на экран кнопку (Buttons -> Button), расположим ее ниже последней текстовой надписи и по центру экрана. Расстояние от последней текстовой надписи до кнопки должно быть 16dp.

  1. Далее нам необходимо изменить идентификатор для наших четырех виджетов, чтобы к ним было удобно обращаться. Для верхней текстовой надписи укажем id под названием label. Выделите верхнюю текстовую надпись на экране редактирования макета или в дереве компонентов, после чего измените атрибут id.

  1. Для поля ввода укажем идентификатор name_et, а для нижней текстовой надписи укажем идентификатор greeting_text. Для кнопки укажем идентификатор greet_button.

Ниже представлен код макета, который мы создали выше.

res/layout/activity_main.xml

<?xml version="1.0" encoding="utf-8"?>
<androidx.constraintlayout.widget.ConstraintLayout 
xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <EditText
        android:id="@+id/name_et"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:ems="10"
        android:inputType="textPersonName"
        android:text="Name"
        app:layout_constraintBottom_toBottomOf="parent"
        app:layout_constraintEnd_toEndOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintTop_toTopOf="parent" />

    <TextView
        android:id="@+id/label"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_marginBottom="8dp"
        android:text="TextView"
        app:layout_constraintBottom_toTopOf="@+id/name_et"
        app:layout_constraintStart_toStartOf="@+id/name_et" />

    <TextView
        android:id="@+id/greeting_text"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_marginTop="16dp"
        android:text="TextView"
        android:textAppearance="@style/TextAppearance.AppCompat.Display1"
        app:layout_constraintEnd_toEndOf="@+id/name_et"
        app:layout_constraintStart_toStartOf="@+id/name_et"
        app:layout_constraintTop_toBottomOf="@+id/name_et" />

    <Button
        android:id="@+id/greet_button"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_marginTop="16dp"
        android:text="Button"
        app:layout_constraintEnd_toEndOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintTop_toBottomOf="@+id/greeting_text" />
</androidx.constraintlayout.widget.ConstraintLayout>

MainActivity.java

public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        EditText name_et = findViewById(R.id.name_et);
        TextView greeting_text = findViewById(R.id.greeting_text);

        Button button = findViewById(R.id.greet_button);
        button.setOnClickListener(v -> {
            String name = name_et.getText().toString();
            String message = getString(R.string.greeting_message, name);
            greeting_text.setText(message);
        });
    }
}
